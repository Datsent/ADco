Send forms directly to Google Sheets, an elementor plugin


== Description ==

Elementor Form Plus allows you to send submitted forms directly from your Wordpress site to Google Sheet and more.

Features include:

* Sending submitted forms from the Elementor forms widget to Google Sheet up to 200 per month In the free version.
* A special plugin in the Elementor panel, which allows you to design cf7 (contact form 7) forms directly from Elementor.
* Send events to Analytics and Facebook on submit.

= Basic usage: =

If you want each form to function as a separate site (e.g. landing page), connect to your google sheet account through connect button on Elementor panel.
If not, go to Dashboard > Elementor Form + and set up your Google Account globally (same account to entire  site).

== Installation ==

= Installing manually: =

1. Unzip all files to the `/wp-content/plugins/elementor-form-plus` directory
2. Log into WordPress admin and activate the 'elementor form plus' plugin through the 'Plugins' menu
3. Go to Elementor
